Who runs Genevieve?

Genevieve is primarily written and maintained by Mad Price Ball, one of the co-founders of Open Humans. Although it uses data from Open Humans, it is a personal project. Mad's work is supported by their funding from the Shuttleworth Foundation.

Genevieve is open source.

Genevieve's source code is shared on GitHub with an MIT (Expat) license.

In addition, notes contributed via Genevieve are public domain (CC0) . Notes are stored in the GenNotes database as public domain material (CC0) and are available via the GenNotes API. The GenNotes server is operated by Open Humans, and the source code for this is also available.)

Deleting and deauthorizing Genevieve

You can go to the account management page to to delete your account from this site, as well as associated genome reports. While this website will no longer retain authorizations for accessing Open Humans, your authorizations will still be available within Open Humans itself. To remove these, you should deauthorize the project in Open Humans. (Do this after deleting the account here!)
